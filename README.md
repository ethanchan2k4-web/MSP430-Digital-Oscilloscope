# MSP430-Digital-Oscilloscope
MSP430 Oscilloscope | C (Embbeded), MATLAB, MSP430 Microcontroller, ADC, UART Feb. 2026
• Designed a low-cost digital oscilloscope capturing 0–3.3V signals using an MSP430 MCU for data acquisition.
• Wrote optimized C code to control ADC10, and UART peripherals, achieving ≈ 326 kSPS on a 16 MHz clock.
• Verified system timing using breakpoints to measure 39 instruction cycles per sample as a hardware limit.
• Plotted signals on MATLAB App with edge triggering, and precise time axis scale to prevent aliasing.
